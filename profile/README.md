# 🥪 OpenFacedSandwich

Welcome to **OpenFacedSandwich** -  
a collection of open-source projects, AI experiments, tools, and slightly questionable ideas…  

**layered like a proper Danish smørrebrød.**

## 🍞 What is this?

OpenFacedSandwich is where we:

- build AI models 🧠  
- create useful tools 🔧  
- run weird experiments 🧪  
- and occasionally do things just because they’re funny 😄  

👉 Everything is open  
👉 Everything is layered  
👉 Nothing is hidden  

## 🥓 The philosophy

Why “OpenFacedSandwich”?

Because:

- **Open-faced** → you see everything → open source  
- **Layers** → how we build software  
- **Stacking** → how things actually work  

👉 We don’t build systems  
👉 we **assemble sandwiches**

## 🍳 How it works

Every project is a **layer**:

- 🍞 Base → core projects  
- 🥩 Protein → AI models  
- 🧅 Toppings → tools & utilities  
- 🌿 Extras → experiments, chaos, and fun  

👉 Combine layers → build something new  
👉 Remove layers → simplify
